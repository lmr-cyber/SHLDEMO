# CLAUDE.md

This is a demo project for SHL exam. Every child module will have its own "README.md" for the requirement.

Code agent should read the requirement for each module generation task. 
This project will use core Java to implement the requirement, it should not be introduced other libraries, and the syntax is based on the JDK version which is use in the requirement.


## Project Type
This is a multiple maven project, the JDK is 21 in root parent setting, but some children modules will specify own JDK version.

**Base Package**: demo.shl.exam
